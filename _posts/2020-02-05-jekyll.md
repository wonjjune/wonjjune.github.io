## 설치과정
* 루비 버전을 확인할 필요가 있고 실해시 번들로 실행해야한다.
1. 루비 설치
2. gem install bundle jekyll
3. bundle install
4. bundle exec jekyll serve
5. post 하위에 포스팅 내용을 작성하며 파일이름은 yyyy-mm-dd-title 형식을 갖추면 되고 내용에 타이틀과 날짜및 카테고리를 지정하여 작성하면 된다.
